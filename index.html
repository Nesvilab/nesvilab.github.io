FragPipe
FragPipe: A simple cross-platform UI for running MSFragger proteomic search engine

Matained By Dmitriy Avtonomov

This is a very basic Java GUI wrapper for MSFragger - ultrafast proteomic search engine. It will help you launch MSFragger for Open and Closed searches and can also run post-processing of results with Peptide/Protein Prophets as well as generate tabular summary results using Philosopher.
